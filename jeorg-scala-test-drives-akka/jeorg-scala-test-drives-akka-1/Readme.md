# jeorg-scala-test-drives-akka-1

## Introduction

In this module, we have a look at the Akka arquitecture.
This module has, as a starting point, the examples provided in the [Akka documentation](https://doc.akka.io/docs/akka/current/typed/guide/tutorial_1.html), under Apache 2.0 licence for version [2.6.15](https://akka.io/blog/news/2021/06/10/akka-2.6.15-released).

## How to run

AkkaQuickstart

```bash
sbt "runMain org.jesperancinha.scala.td.akka1.AkkaQuickstart"
```

Actor Hierarchy experiments

```bash
sbt "runMain org.jesperancinha.scala.td.akka1.ActorHierarchyExperiments"
```
